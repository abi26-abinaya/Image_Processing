# 🖼️ Image Processing Web App

A simple and beginner-friendly web application to upload and process images using Python, Flask, and OpenCV. Users can apply filters like **grayscale** and **edge detection** and preview results directly in the browser.

---

## 🔧 Features

- 📤 Upload image files (PNG, JPG, etc.)
- 🎨 Apply image filters:
  - Grayscale
  - Edge Detection (Canny)
- 🖥️ Real-time output preview
- 💡 User-friendly and responsive frontend
- 🔙 Reset and re-upload support

---

## 💻 Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python, Flask  
- **Image Processing:** OpenCV
